# TweetyPy
Machine learning app that can create own tweets and word clouds by learning from user statuses or top trend tweets using Markov chain. This is bot's
[Twitter profile](https://twitter.com/_TweetyPy_) <br /><br />

## Installation

Clone this repo and install requirements.txt
```
$ git clone https://github.com/SerhiiStets/TweetyPy.git
$ pip install -r /path/to/requirements.txt
```
